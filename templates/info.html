How does it work?

Using a list of existing names, the algorithm generates new unique Khajiit names that can be used for Role-play. More races might come in the future. The source of the Khajiit names sample comes from this website . If you want to work on the website or contact me, you can find the source code on Github: https://github.com/jaypey/wnamegen.
